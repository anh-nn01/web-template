Projects

Physics-Guided Reinforcement Learning System for Realistic Vehicle Active Suspension Control

Designed and implemented a Reinforcement Learning agent (DDPG) that learn to control the car quarter suspension so that the vertical movements of the car's body is minimizede, ensuring a comfort ride experience for passengers. Unlike previous work, our work explicitly consider the physical contraints of the actual mechanical system and train the agent to actively control the suspension in real-time under such real constraints. Quantitatively, our algorithm outperforms passive systems in reducing the average vehicle body velocity and acceleration by 43.58% and 17.22%, respectively, minimizing the vertical movement impacts on the passengers.

Generate Map from Satellite Imagery using Generative Adversarial Network

Designed and implemented Pix2Pix, a general Deep Learning framework for image translation task, specifically for translating satellite imagery to equivalent map.

The algorithm takes any satellite image as the input, run inference through its U-Net backbone, and output an image of the same size with equivalent map representation.

AI Lunar Lander

Created an AI agent that use Double Deep Q-learning in Reinforcement Learning to learn by itself how to land a Lunar Lander in OpenAI Gym environment. With optimization through trial-and-error, the agent successfully solved the game after about 500 games, landing the lundar lander to the target on the Moon's surface safely and quickly.
